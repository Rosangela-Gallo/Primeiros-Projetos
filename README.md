# Primeiros-Projetos
O projeto ETL foi construído usando   usando  para  anãlise de Imóveis de Aluguel  os dados da  empresa  Airbnb  New York,  disponibilizados de  aberta.
Foi construído com a ferramenta Google Colab e utilizando principalmente a linguagem python com pandas e pyspark
O arquivo, disponibilizado no formato CSV conta com aproximadamente 48mil registros e teve sua base e finalização disponibilizados na plataforma
GCP utilizando o CloudSorage, o BigQuery e o DataStudio.
